Hadoop-MapReduce-示例

发表于2024-01-31|更新于2024-06-29|Hadoop
|阅读量:

其实就是要重写 mapper 、reducer 和 driver 三个类

WordCount

Mapper

主要是重写 map 方法，对于进入的每行数据都会被执行，map 方法

Reducer

其实就是对 Mapper 对输出进行聚合，需要重写 reduce 函数

Driver

负责启动和连接 Mapper 与 reducer。编写的流程是固定的。

序列化

因为数据需要在服务器之间进行传输与聚合，而当 Key 或者 Value 的数据类型不是已经被定义的基本类型时（例如一个 Value 由多个基本类型组成），就无法传输，所以需要对这个 Value 创建一个类，并使其能够序列化。
