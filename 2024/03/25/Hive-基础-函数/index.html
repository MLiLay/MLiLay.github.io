substring

    replace

replace (string A, string b, string c)

A 中 b 字符串替换为 c 字符串

regexp_replace 正则替换

regexp_replace (string A, string b, string c)

将字符串 A 中符合正则表达式 b 的部分替换为字符串c

regexp 正则匹配

若字符串符合正则表达式返回 true，否则 false。

repeat (string A, int n)

返回array

concat

日期函数

unix_timestamp

返回当前或指定时间的时间戳

from_unixtime

转化 UNIX 时间戳（从 1970-01-01 00:00:00 UTC 到指定时间的秒数）到当前时区的时间格式

current_timestamp

当前时间，精确到单位 ms

month

获取日期中的月

day

获取输入日期中的天

hour

获取当前输入日期中的小事

datediff

两个日期的相差天数

datediff (string endate, string startdate)

返回 int

date_add

日期加天数

date_add (string startdate, int days)

date_sub

日期减天数

date_sub (string startdate, int days)

data_format

将标准日期解析成指定格式字符串

  • case a when b then c when d then e else f end

if

size

集合中的元素个数

map

map (key 1, value 1, key 2, value 2, …)

map_keys

返回输入 map 中的keys

map_values

返回 map 中的values

array

array (val 1, val 2, …)

array_contains

判断输入的 array 中是否包含元素

sort_array

将 array 中的元素排序

struct

声明 struct 中的属性值

named_struct

声明 struct 中的属性和值

高级聚合函数

collect_list

收集并返回一个list

collect_set

收集并放回一个set

炸裂函数（UDTF）

其实就是将本来一列中的一个集合中的元素进行拆分出来。

窗口函数
分类 函数名称 语法 说明 支持SQL 支持SPL
聚合函数 聚合函数 无 所有聚合函数都支持在窗口函数中使用。聚合函数列表请参见聚合函数。 √ ×
排名函数 cume_dist函数 cume_dist() 统计窗口分区内各个值的累计分布。即计算窗口分区内值小于等于当前值的行数占窗口内总行数的比例。返回值范围为(0,1]。 √ ×
dense_rank函数 dense_rank() 窗口分区内值的排名。相同值拥有相同的排名，排名是连续的，例如有两个相同值的排名为1，则下一个值的排名为2。 √ ×
ntile函数 ntile(_n_) 将窗口分区内数据按照顺序分成N组。 √ ×
percent_rank函数 percent_rank() 计算窗口分区内各行的百分比排名。 √ ×
rank函数 rank() 窗口分区内值的排名。相同值拥有相同的排名，排名不是连续的，例如有两个相同值的排名为1，则下一个值的排名为3。 √ ×
row_number函数 row_number() 窗口分区内值的排名。每个值拥有唯一的序号，从1开始。三个相同值的排名为1、2、3。 √ ×
偏移函数 first_value函数 first_value(_x_) 返回各个窗口分区内第一行的值。 √ ×
last_value函数 last_value(_x_) 返回各个窗口分区内最后一行的值。 √ ×
lag函数 lag(_x_, _offset_, _default_value_) 返回窗口分区内位于当前行上方第_offset_行的值。如果不存在该行，则返回_default_value_。 √ ×
lead函数 lead(_x_, _offset_, _default_value_) 返回窗口分区内位于当前行下方第_offset_行的值。如果不存在该行，则返回_default_value_。 √ ×
nth_value函数 nth_value(_x_, _offset_) 返回窗口分区中第_offset_行的值。 √ ×

聚合函数

当 function 为聚合函数是可以使用frame

frame：{ rows | range} { frame_start | frame_between }

例如range between unbounded preceding and unbounded following
